Tags: <h1> ... <h6> e <p>

(voltar)

A tag <h1> até <h6> serve para títulos, quanto o maior o número da junto do h "menos importante" é o título, gerando assim o subtítulos.

Já a tag <p> define os paragrafos dentro do texto.

Tags: <ol>, <ul> e <li>

(voltar)

Tanto <ol> quanto <ul> geram listas, sendo <li> usada para definir os itens dessas listas, contudo <ol> gera listas ordenadas enquanto o <ul> não.

Código:

Tags: <strong>, <i> e <u>

(voltar)

A tag <strong> deixa o texto em negrito, assim como a tag <b>[1], e serve para dá importância a uma palavra ou frase.

A tag <i> é usada para dar destaque a um texto ou frase, geralmente por ser um termo técnico, expressão ou outro idioma. Possui um efeito em itálico, assim como a tag <em> que no caso é usada para dar ênfase no texto[2].

A tag <u> antes era conhecida como a tag underline, sendo depreciada no HTML4[3] e retornando no HTML5 como a tag unarticulated, ou seja, com o sentido definido para uso de palavras escritas erroneamente.

Tags: <mark>, <sub> e <sup>

(voltar)

A tag <mark> serve para dá relevância a um texto com um efeito de marca-texto/background-color.

A tag <sub> cria um texto abaixo da "linha", por exemplo: H2O - o "2" está usando a tag.

A tag <sup> cria um texto acima da "linha", por exemplo: A2 = B2 + C2

[4]

Tags: <blockquote>, <del> e <abbr>

(voltar)

A tag <blockquote> cria uma citação grande, enquanto uma <q> pode ser usada para citações pequenas - maior diferenças entre elas é que a primeira dá um espaçamento e recuo, enquanto a segunda usa somente aspas[5].

A tag <del> serve para marca parte do texto que se tornou irrelevante como esse exemplo

A tag <abbr> serve para designar abreviações, exemplo (passe o mouse na palavra): ONU

Tags: <a>, <hr> e menção honrosa <font>

(voltar)

A tag <a> contém o atributo href com ele é possível criar hiperlinks, no qual se pode acessar outras páginas ou baixar arquivos entre várias outras coisas[6].

A tag <hr> define um quebra entre conteúdo.

A tag <font> servia para definir uma fonte, cor e tamanho, porém acabou sendo depreciada no html5[7], sendo necessário fazer isso diretamente com css.
